# Prompt 02 — Fountain Script Generation

## AI Tool / Model
Cursor Agent (Claude)

## Intended Output
A Fountain-format screenplay defining 6 scenes with concrete visual actions, on-screen text, and voiceover lines.

## Prompt Text

```
Write a Fountain script for a 60-second marketing video for "Kavana" — an AI deep-work companion.

Follow "Show, Don't Tell":
- BAD: "The user feels overwhelmed"
- GOOD: "Notification cards cascade across the screen — Slack, email, calendar — each pulsing red. A focus meter drains from 100% to 12%."

Structure (6 scenes, ~10s each):
1. HOOK — Bold question, brand name appears
2. PROBLEM — Notification cascade, fractured calendar, draining meter
3. SOLUTION — Kavana shield activates, noise fades
4. FEATURES — Three beats: Focus Shield, Smart Orchestration, Clarity Dashboard
5. PROOF — Before/after stats animation
6. CTA — "Start your focused day" + Hebrew tagline

Include: scene headings, visual action lines, on-screen text, voiceover narration.
Tone: confident, calm, modern tech.
Bilingual elements where natural (Hebrew tagline in final scene).
```

## Outcome
**Accepted with minor edits** — Script saved to `script/script.fountain`. Shortened voiceover in Scene 4 for timing.

## Iteration Notes
Initial draft had 7 scenes; consolidated to 6 to hit 60s target cleanly.
