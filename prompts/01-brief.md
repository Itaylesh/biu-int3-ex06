# Prompt 01 — Business Brief Generation

## AI Tool / Model
Cursor Agent (Claude)

## Intended Output
A concise marketing/business brief for the fictional AI product "Kavana" to guide script, JSON, and Remotion implementation.

## Prompt Text

```
Create a business brief for a fictional AI product called "Kavana" (Hebrew: כוונה = intention/focus).

Product: AI deep-work companion that shields users from digital distraction and auto-orchestrates their day.

Requirements:
- Target audience: knowledge workers 22-35
- Core problem: notification overload, fragmented attention, manual calendar juggling
- Three features: Focus Shield, Smart Orchestration, Clarity Dashboard
- Visual style: modern corporate, blue-violet palette, clean motion design
- Video: 60-second explainer, 6 scenes (hook, problem, solution, features, proof, CTA)
- Bilingual: English primary, Hebrew tagline "כוונה אחת. יום שלם."
- All visuals must be code-generated in Remotion (no external image APIs)
- Soundtrack via Suno (calm piano + synth, rising energy)

Output: problem statement, solution, target audience, key messages, scene outline, visual tone notes.
Use "Show, Don't Tell" — concrete visual descriptions, not vague emotions.
```

## Outcome
**Accepted** — Brief integrated into PRD.md and scene design. Minor edits: renamed tagline emphasis and confirmed 6-scene structure.

## Iteration Notes
No rejection. Brief served as direct input for script.fountain and scenes.json generation.
